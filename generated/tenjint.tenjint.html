tenjint.tenjint¶

Description

tenjint’s main module.

This is tenjint’s main module that can be used to initialize, uninitialize, and to run tenjint.

Functions

init(configs)

Initialize tenjint.

run([configs])

Initialize tenjint, start the event loop, and uninitialize tenjint after it returns.

uninit()

Uninitialize tenjint.
